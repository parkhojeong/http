- Cache-Control(Response)
cache directive: client cache management rules 
  - max-age: valid time, unit: second
  - no-cache: use cache always after check from server(origin)
  - no-store: never cache

- Pragma(backward compatibility)
    - no-cache
    - for HTTP/1.0
    
- Expires(backward compatibility)
  - DateTime
  - after Http/1.0 
  - max-age recommended
  - priority: max-age > Expires

Validator and conditional request header
- Validator
  - ETag: "v1.0", "hash value"
  - Last-Modified: DateTime
- conditional request header
  - If-Match, If-None-Match: ETag
  - If-Modified-Since, If-Unmodified-Since: Last-Modified
