- Last-Modified, If-Modified-Since cons
  - unit is second
  - based date
  - same data, different time(delete and restore) - Last-Modified updated
  - server can't manage cache

- ETag, If-None-Match
  - server manage cache
  - ex. versioning, hash
  - only compare ETag(simple)
  

== ETag example start==
[request]
GET /star.jpg HTTP/1.1
Host: www.example.com

[response]
HTTP/1.1 200 OK
Cache-Control: max-age=60 
ETag: aaa
Content-Type: image/jpeg
Content-Length: 10000

( image data )
[request after 60seconds]
GET /star.jpg HTTP/1.1
If-None-Match: aaa
Host: www.example.com

[response]
HTTP/1.1 304 Not Modified
Cache-Control: max-age=60
ETag: aaa
Content-Type: image/jpeg
Content-Length: 10000

[client use browser cache]
