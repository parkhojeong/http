[HTTP message structure]
start-line
*(header-field CRLF)
empty line(CRLF)
[message-body]

https://tools.ietf.org/html/rfc7230#section-3.1
[HTTP REQUEST structure]
request-line
*(header-field CRLF)
CRLF
[message-body]

[HTTP RESPONSE structure]
status-line
*(header-field CRLF)
CRLF
[message-body]

- request-line = METHOD /absolute-path[?query] HTTP/version
- status-line = HTTP/version SP status-code SP reason-phrase CRLF 
- header-field = field-name ":" OWS field-value OWS
- message-body = handle all byte data

GET / HTTP/1.1
Host: www.example.com
