# data transfer method
1. query parameter - GET
2. message body - POST, PUT, PATCH


# 4 situations
1. query static data - GET without query parameter
GET /static/star.jpg HTTP/1.1
HOST:https://www.google.com/

2. query dynamic data: GET with query parameter
GET search?q=hello HTTP/1.1
HOST:https://www.google.com/

3. HTML form
- post(x-www-form-urlencoded) with message body
- get with query parameter
- support get, post
[3.1 method="post"]
- data UTF-8 encoding
- default application/x-www-form-urlencoded
== HTML form ==
<form action="/save" method="post">
    <input type="text" name="username"/>
    <button type="submit">submit</button>
</form>

== HTTP message ==
POST /save HTTP/1.1
HOST:https://www.google.com/
Content-Type:application/x-www-form-urlencoded

username=kim&age=20

[3.2 method="get"]
== HTML form ==
<form action="/members" method="get">
    <input type="text" name="username"/>
    <input type="text" name="age"/>
    <button type="submit">submit</button>
</form>

== HTTP message ==
GET /members?username=kim&age=20 HTTP/1.1
HOST:https://www.google.com/

[3.3 enctype="multipart/form-data"]
- file upload
== HTML form ==
<form action="/save" method="post" enctype="mutlipart/form-data">
    <input type="text" name="username"/>
    <input type="file" name="file1"/>
    <button type="submit">submit</button>
</form>

== HTTP message ==
POST /save HTTP/1.1
HOST:https://www.google.com/
Content-Type:multipart/form-data; boundary=---XXX
Content-Length:123

---XXX
Content-Disposition:form-data; name="username"

kim
---XXX
Content-Disposition:form-data; name="file1"; filename="hello.png"
Content-Type:image/png

1092389...
---XXX--


4. HTTP API
- GET, POST, PUT, PATCH, DELETE
- application/json is de facto

== HTTP message ==
POST /members HTTP/1.1
HOST:https://www.google.com/
Content-Type:application/json

{
    "username":"kim",
    "age":20
}
