URI(Uniform Resource Identifier)

origin: https://www.ietf.org/rfc/rfc3986.txt

URI( Uniform Resource Identifier)
- URL( Uniform Resource Locator) 
- URN( Uniform Resource Name)

foo://example.com:8042/over/there?name=ferret#nose
- foo: scheme
- example.com:8042: authority
- /over/there: path
- ?name=ferret: query
- #nose: fragment

urn:example:animal:ferret:nose
- scheme: urn
- example:animal:ferret:nose: path

URI
Uniform: unified method of describing a resource
Resource: identifiable things by URI
Identifier: required information to identify a resource

URL
 - Locator: address of a resource
 - mostly the same as URI
URN 
 - Name: unique name for a resource 
 - not commonly used

URL 
- https://www.google.com/search?q=hello&hl=ko

URL grammar
- scheme://[userinfo@]host[:port][/path][?query][#fragment]
- https://www.google.com:443/search?q=hello&hl=ko
 - scheme: protocol(https)
 - host: www.google.com
 - port: 443
 - path: /search
 - query: q=hello&hl=ko

scheme
- mostly used protocol
- http, https, ftp, etc
- http(80), https(443)

host
- domain name or IP address

port
- omission: https(443), http(80)

path
- resource path
- hierarchical structure
- ex
 - /home/file1.jpg
 - /members
 - /items/iphone12

query
- key=value
- start with "?"
- & separate parameters
- called query string, query parameter

fragment
- anchor
- start with "#"
