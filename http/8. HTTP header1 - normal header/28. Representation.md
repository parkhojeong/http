Representation
- Content-Type: data format
- Content-Encoding: compression method
- Content-Language
- Content-Length: byte size


Content-Type
- media type, character encoding
- ex. 
 - text/html; charset=utf-8;, application/json;, image/jpeg;

Content-Encoding
- compression method
- ex. gzip, deflate, identity

Content-Language
- language
- ex. ko, en, en-US

Content-Length
- byte size
- can't use it with Transfer-Encoding
