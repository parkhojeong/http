Content-Negotiation
: prefer representation for client(Request Header) 
- Accept: media type
- Accept-Charset: encoding
- Accept-Encoding: compression encoding
- Accept-Language: language  

[Quality value]
1. higher q is priority
- empty = 1st preference
- Quality value(q) 0~1
- Accept-Language: ko-KR,ko;q=0.9,en-US;q=0.8,en;q=0.7
  - ko-KR;q=1 (q is empty)
  - ko;q=0.9
  - en-US;q=0.8
  - en;q=0.7

2. specific type is prior
- Accept: text/*, text/plain, text/plain;format=flowed, */* 
  - 1. text/plain;format=flowed
  - 2. text/plain
  - 3. text/*
  - 4. */*

3. match specific type
- Accept: text/*;q=0.3, text/html;q=0.7, text/html;level=1,
- text/html;level=2;q=0.4, */*;q=0.5
    - text/html;level=1 -> quality 1
    - text/html;level=2 -> quality 0.4
    - text/html;level=3 -> quality 0.7
    - text/html -> quality 0.7
    - text/plain -> quality 0.3
    - text/jpeg -> quality 0.5
