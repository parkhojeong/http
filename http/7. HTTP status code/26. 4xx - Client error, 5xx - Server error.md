4xx(Client Error)
- client mistake

400 Bad Request
ex. requests parameter, json format error

401 Unauthorized
- need authentication 
- WWW-Authenticate header

403 Forbidden
- insufficiency authorization 

404 Not Found
- resource not found
- hide resource for client without authorization

5xx(Server Error)

500 Internal Server Error(mostly used)

503 Service Unavailable
- temporarily server busy or maintenance
- can use Retry-After header
