- transport layer: TCP, UDP 
- internet layer: IP
- network interface layer

tcp(Transmission Control protocol) feature
- connection oriented (3 way handshake)
 - virtual connection(logical connection)
- data delivery guarantee
- order guarantee

3 way handshake
- client send SYN
- server send SYN+ACK
- client send ACK

udp(user datagram protocol)
- port number, checksum
- simple and fast
